Las Colas:
-Los id sucesores de las colas solo apuntan a ids de combis.
-Los id Antecesores no estan en el arreglo de ids de colas.

Las combis:
-Los id Antecesores, nunca 0, solo estan en el arreglo de ids de colas.
-Los id sucesores nunca apuntan a ids del arreglo de las combis.
-nunca ambos 0.

Las normales y funciones:
-Los id sucesores nunca apuntan a ids del arreglo de las combis , nunca 0.
-Los id Antecesores no estan en el arreglo de ids de colas, nunca 0.

Las contadores:
-Los id sucesores nunca apuntan a ids del arreglo de las combis ni de contadores.
-Los id Antecesores no estan en el arreglo de ids de colas ni de contadores, nunca 0.
